# N3: Inverted Pyramid News Writing

## Objective
Write visa news articles using the journalistic inverted pyramid. The most critical information comes first, followed by supporting details, then background context. This ensures readers and AI extractors always get the key facts — even if they only read the first paragraph.

## The Inverted Pyramid Structure

```
┌─────────────────────────────────────────┐
│   LEAD (5Ws1H)                          │  ← Most important
│   Who, What, When, Where, Why, How      │
├─────────────────────────────────────────┤
│   CRITICAL DETAILS                       │
│   Key numbers, effective dates,          │
│   who is affected, what changes          │
├─────────────────────────────────────────┤
│   CONTEXT & BACKGROUND                   │
│   Previous policy, reason for change,    │
│   historical context, expert comment     │
├─────────────────────────────────────────┤
│   INDIAN-SPECIFIC IMPACT                 │
│   How this affects Indian applicants,    │
│   INR equivalents, India-specific steps  │
├─────────────────────────────────────────┤
│   RELATED INFORMATION                    │
│   Links to guides, FAQ snippets,         │
│   official sources, further reading      │
└─────────────────────────────────────────┘
```

## News Article Template

```markdown
# [Headline: Must include country + visa type + key change + "for Indians"]

[Lead Paragraph — MAX 3 sentences]
[Country] has [announced/changed/updated/implemented] [specific policy change]
for [visa type] visas, effective [date]. This means [impact in simple terms]
for Indian applicants. [Key number or fact that matters most].

[Critical Details — 2-4 paragraphs]

**What changed:**
[Specific policy change description in plain language]

**Effective date:**
[Date] — [immediate/future/scheduled]

**Who is affected:**
- [Category 1: e.g., New applicants]
- [Category 2: e.g., Existing applicants]
- [Category 3: e.g., Specific groups like students/workers]

**Key numbers:**
- Old: [Previous fee/requirement/threshold]
- New: [Updated fee/requirement/threshold]
- Difference: [INR equivalent if applicable]

[Context & Background — 1-3 paragraphs]

[Paragraph explaining previous policy/rules]
[Paragraph on reason for change if known]
[Expert or official quote if available — attributed]

[Indian-Specific Impact — 1-2 paragraphs]

**What this means for Indian applicants:**

For Indian students/workers/tourists applying for [visa type], this change
means [specific impact]. 

[Additional India-specific detail: VFS processing, INR costs,
document requirements from India, etc.]

[Related Information]
- [Link to official source]
- [Link to relevant BTWVisas guide]
- [FAQ snippet or quick tip]
```

## News Article Types & Templates

### Type 1: Policy Change

```
Headline: "UK Increases Student Visa Financial Requirement for Indians — Effective June 2026"

Lead: The UK Home Office has raised the minimum financial requirement for
Student Visas from £1,334/month to £1,483/month (approx. ₹1.6 lakhs),
effective 1 June 2026. Indian students must now show proof of
£14,847 (₹16.1 lakhs) for a 9-month course.

Body: [Details on the change, who it applies to, exceptions]
Indian Impact: [INR comparison, approved banks in India, document tips]
```

### Type 2: New Visa / Program Launch

```
Headline: "Canada Launches New Tech Talent Visa for Indians — Key Details"

Lead: Immigration, Refugees and Citizenship Canada (IRCC) has announced a
new Tech Talent Visa pathway for Indian professionals in AI, cybersecurity,
and software development. The program opens on [date] with a streamlined
2-week processing time.

Body: [Eligibility, quota, application process]
Indian Impact: [How Indians can apply, document requirements, success rate]
```

### Type 3: Fee Change

```
Headline: "Schengen Visa Fee Hiked for Indians — New Rates from [Date]"

Lead: The Schengen states have increased the visa application fee for
Indian passport holders from €80 (₹7,200) to €90 (₹8,100), effective
[date]. The increase applies to all short-stay Schengen visa applications.

Body: [Fee table comparison, who is exempted, additional service charges]
Indian Impact: [Total cost including VFS fees, INR breakdown]
```

### Type 4: Processing Time Update

```
Headline: "US Visa Processing Time Reduced for Indian Applicants — New Timeline"

Lead: The US Embassy in India has reduced visitor visa (B1/B2) processing
time from 200 days to 120 days, effective immediately. Wait times for
interview appointments have also been reduced in key cities.

Body: [New timelines table, city-wise breakdown, reasons]
Indian Impact: [Best cities for appointments, premium processing options]
```

## Writing Rules

### Rule 1: Lead Paragraph Must Stand Alone
```
✓ "The UK Home Office has raised the minimum financial requirement for
   Student Visas from £1,334/month to £1,483/month (approx. ₹1.6 lakhs),
   effective 1 June 2026."

✗ In the latest update, the UK government has made changes to their
   student visa program that will impact applicants. The new rules
   introduce updated financial requirements that applicants must meet.

→ The first lead gives ALL key facts in 1 sentence. The second is vague
  and forces readers to dig for answers. News leads must NOT be vague.
```

### Rule 2: Use Indian Units & Context
```
✓ "The fee increases from €80 (₹7,200) to €90 (₹8,100)"
✓ "Applicants must show £14,847 (approximately ₹16.1 lakhs)"
✓ "Apply at any of the 12 VFS centers across India including Delhi, Mumbai, Bengaluru"

✗ "The fee increases from €80 to €90"
✗ "Applicants must show £14,847"
```

### Rule 3: Attribute Everything
```
✓ "According to the UK Home Office press release dated [date]..."
✓ "The Indian Embassy in Paris confirmed via Twitter..."
✓ "VFS Global published an updated fee schedule showing..."

✗ "It is reported that visa fees are increasing..."
✗ "Sources indicate that processing times will change..."
→ Every claim must have a named, verifiable source.
```

### Rule 4: Keep Paragraphs Tight
```
✓ Max 3 sentences per paragraph
✓ Max 60 words per sentence
✓ Bullet points for lists of changes
✓ Tables for fee comparisons
✓ Bold key numbers and dates
```

### Rule 5: No Opinion, No Speculation
```
✓ "The policy change means Indian applicants must now show additional funds."
✗ "This unfortunate change will make it harder for Indian students to study abroad."
✗ "We think this is a positive step for the visa system."
→ Let the facts speak. Save analysis for a separated "Expert Take" section if needed.
```

## Word Count Guidelines

| Section | Breaking News | Standard Article |
|---------|--------------|-----------------|
| Headline | 12-18 words | 12-18 words |
| Lead paragraph | 50-80 words | 50-80 words |
| Critical details | 150-250 words | 200-400 words |
| Context/background | 100-150 words | 200-300 words |
| Indian Impact | 80-120 words | 150-250 words |
| **Total** | **400-600 words** | **600-1,000 words** |

## Tone & Voice

| Attribute | News Tone |
|-----------|-----------|
| Primary | **Journalistic** — neutral, factual, attributed |
| Secondary | **Helpful** — Indian context, practical implications |
| Avoid | Sensationalism, clickbait, speculation, promotional language |
| Audience | Indian travelers/applicants — assume no prior knowledge of the policy |

## 🔔 Data Change Flag — Triggers Guide Update Alert

When writing ANY article, check if the content contains these data points.
If YES, this article MUST trigger a 🔔 guide update alert via N6.

| Data Point in Article | Alert | Example |
|----------------------|-------|---------|
| **Fee/amount change** | 🔴 ALERT | "Fee increased from €80 to €90" → update fee table in guide |
| **Processing time change** | 🔴 ALERT | "Processing reduced from 200 to 120 days" → update timeline in guide |
| **Eligibility requirement change** | 🔴 ALERT | "Maintenance amount changed from X to Y" → update criteria section |
| **Document requirement change** | 🟡 ALERT | "New bank statement rules" → update document checklist |
| **Effective date/deadline** | 🟡 ALERT | "New rules effective June 1" → update "Latest Updates" section |
| **Visa category launch/removal** | 🟡 ALERT | "New Tech Talent visa announced" → add new section to guide |

**Rule:** If any of these boxes are checked during writing → N6 alert is MANDATORY, not optional.

## 🧠 Daily "Find News" Workflow Integration

When you say **"find news"** (N0 orchestrator), this is the CORE writing skill:

```
For each 🔴/🟡 article selected in N2 triage:

1. I pick the right template:
   - Policy change → Type 1
   - New visa/program launch → Type 2
   - Fee change → Type 3
   - Processing time update → Type 4

2. I write using inverted pyramid:
   - Lead paragraph = ALL critical facts (standalone)
   - Critical details = key numbers, dates, who's affected
   - Indian impact = INR, VFS, India-specific steps
   - Target: 400-600 words (breaking) or 600-1,000 words (standard)

3. I cite sources inline per N1 rules
4. I keep each article under 30 minutes to write
```

**Time spent in daily workflow:** ~20-30 min per article (3-5 articles = 1-2.5 hours total)

## Output
A complete news article in markdown format with:
- Inverted pyramid structure
- All critical facts in the lead paragraph
- Indian-specific impact section
- Attributed sources throughout
- "What This Means for Indians" clarity
- Ready for schema markup (see N5)

- [ ] All claims attributed to named, verifiable sources
- [ ] Indian-impact section included (INR conversions, India-specific steps)
- [ ] No speculation or opinion
- [ ] No sentence longer than 60 words
- [ ] No paragraph longer than 3 sentences
- [ ] Headline under 18 words with key facts
- [ ] Source URLs included for verification
