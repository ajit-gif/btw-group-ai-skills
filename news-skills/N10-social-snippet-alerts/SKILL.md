# N10: Social Snippet & Breaking News Alert Crafting

## Objective
Craft platform-optimized snippets for every news article to drive traffic, build authority, and alert Indian travelers to important visa changes. Each platform has its own format, character limits, and audience expectations.

## Platform-Specific Templates

### 1. Twitter/X

| Element | Spec |
|---------|------|
| Character limit | 280 (aim for 200-250 to allow retweets with comments) |
| Image | Single image (16:9, 1200×675px) or infographic |
| Hashtags | Max 2-3 relevant: #VisaUpdate #VisaNews #CountryName |
| Link | Always include (use bit.ly or native shortener if needed) |
| Tone | Urgent, informative, neutral |

**Template — Breaking News:**
```
🚨 BREAKING: [Country] [Action] [Visa Type] for Indians

[Key fact — what changed, effective when]

[1-line impact on Indian applicants]

🔗 [URL]
#VisaUpdate #[Country]
```

*Example:*
```
🚨 BREAKING: UK raises Student Visa maintenance requirement to £1,483/month (₹1.6 lakhs)

Indian students must now show £14,847 for 9-month courses. Effective June 1.

🔗 btwvisas.com/visa-news/uk-student-visa-fee-2026
#VisaUpdate #UKVisa #StudyAbroad
```

**Template — Standard News:**
```
📰 [Country] [announces/launches/changes] [visa type] for Indians

[Key detail — 1 sentence]

[1-line impact]

🔗 [URL]
#VisaNews #[Country]
```

### 2. WhatsApp Broadcast / Status

| Element | Spec |
|---------|------|
| Character limit | No hard limit, but keep under 500 for readability |
| Format | Plain text with line breaks (no markdown in broadcast) |
| Image | Optional — use infographic or news image |
| Tone | Direct, personal, helpful |
| Frequency | Max 2-3 per week (don't spam) |

**Template:**
```
📰 VISA UPDATE: [Country] — [Visa Type]

[What changed in 1-2 sentences]

🕐 Effective: [Date]
🇮🇳 Impact on Indians: [1-2 sentences]
🔗 Read full details: [URL]

— BTWVisas
```

*Example:*
```
📰 VISA UPDATE: UK — Student Visa

UK has increased the monthly maintenance requirement from £1,334 to £1,483 for student visa applicants.

🕐 Effective: June 1, 2026
🇮🇳 Impact: Indian students need to show ~₹1.45 lakhs extra for a 9-month course
🔗 Read full details: btwvisas.com/...

— BTWVisas
```

### 3. Telegram Channel

| Element | Spec |
|---------|------|
| Character limit | No limit, but prefer concise (300-600 chars) |
| Format | Supports markdown: **bold**, *italic*, `code`, [links](url) |
| Image/File | Attach image, PDF, or infographic |
| Buttons | Can add inline buttons for links |
| Tone | Informative, community-oriented |

**Template:**
```
**📰 [Country] [Visa Type] Update — [Date]**

**What changed:**
[1-2 sentences]

**Effective:** [Date]
**Impact on Indians:** [1-2 sentences]

**Key numbers:**
• Old: [value]
• New: [value]
• Difference: [INR amount]

[Read more](URL)
```

### 4. Email Alert (Newsletter)

| Element | Spec |
|---------|------|
| Subject line | 40-60 characters, include key fact |
| Preheader | 80-100 characters, expand on subject |
| Structure | Hero section → Summary → Details → CTA |
| Image | Hero image (1200×675px) |
| Tone | Helpful, authoritative |
| Frequency | Weekly digest + breaking alerts (🔴 only) |

**Subject Line Templates:**
```
✓ Breaking: UK Student Visa — New Financial Requirement from June 1
✓ Canada Launches Tech Talent Visa for Indians — Apply Now
✓ Schengen Visa Fee Hike — What Indians Need to Know

✗ Visa Update Newsletter — May 2026 (too vague)
✗ Important News for Visa Applicants (clickbaity)
```

**Email Body Template:**
```html
Subject: [Breaking/Weekly] Visa Update: [Key Change]

🔴 BREAKING VISA NEWS

[Country] has [announced/changed] [visa type]

[1-2 sentence summary]

---

🇮🇳 WHAT THIS MEANS FOR INDIANS

[Indian-specific impact analysis]

👉 Read the full article: [URL]

---

📌 RELATED GUIDES
- [Guide 1]: [URL]
- [Guide 2]: [URL]

---
BTWVisas — Helping Indians navigate visa applications
[Unsubscribe] | [Update Preferences]
```

### 5. Push Notification

| Element | Spec |
|---------|------|
| Title | 30-40 characters |
| Body | 60-80 characters |
| Image | Optional (120×120px icon) |
| Action | Opens the article URL |
| Frequency | 🔴 Critical news only (max 2-3 per month) |

**Template:**
```
Title: 🚨 UK Student Visa Update
Body: New financial requirement: £1,483/month — effective June 1
```

## Platform Posting Schedule

| Platform | 🔴 Breaking | 🟡 Standard | 🟢 Digest |
|----------|-------------|-------------|-----------|
| Twitter/X | Within 1 hour | Same day | Day of publish |
| WhatsApp | Within 2 hours | If high impact | Digest only |
| Telegram | Within 1 hour | Same day | Day of publish |
| Email | Within 4 hours | Weekly roundup | Weekly roundup |
| Push | Within 1 hour | Not recommended | Not recommended |

## Snippet Writing Rules

### Rule 1: Lead with the News, Not the Platform
```
✓ "UK raises student visa maintenance to £1,483/month"
✗ "New blog post: UK student visa update"
✗ "Check out our latest article on..."
```

### Rule 2: One Key Fact Per Snippet
```
✓ "Indian students need to show £1,483/month (₹1.6 lakhs) — up from £1,334"
✗ "UK changed student visa rules including maintenance, English language, and work rights"
→ Pick the MOST important fact for each platform. Link for details.
```

### Rule 3: Include Indian Impact
```
✓ Every snippet must answer "What does this mean for Indians?"
✓ Use INR where applicable
✓ Use India-specific terms (VFS, center names, etc.)
```

### Rule 4: Image Best Practices
```
✓ Use the same 1200×675px feature image across platforms
✓ Resize for platform-specific requirements
✓ No text overlay on images (hurts Google News/Discover)
✓ For Twitter, consider a simple text-free infographic with key numbers
```

### Rule 5: Cross-Platform Consistency
```
✓ Same key facts across all platforms
✓ Same or similar headline/angle
✓ Consistent branding (BTWVisas)
✓ All snippets link to the same article URL
```

## Snippet A/B Testing

| Element to Test | Variant A | Variant B |
|-----------------|-----------|-----------|
| Tone | Urgent: "BREAKING: UK Raises Visa Fees" | Informative: "UK Updates Student Visa Financial Rules" |
| Format | Text only | Text + image |
| Length | Short (150 chars) | Long (250 chars) |
| Hashtags | 1 hashtag | 3 hashtags |

Track: Click-through rate per platform. Optimize based on 4-week data.

## Social Correction Template

When a correction is published (see N8), post on ALL platforms:

```
⚠️ CORRECTION: Our earlier post on [topic] contained an error.

Correct: [Correct fact]
Incorrect: [Incorrect fact that was posted]

We apologize. Full correction: [URL]
```
