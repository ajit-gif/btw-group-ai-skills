# B09: Blog → Social Media Cross-Post Snippets

## Objective
Repurpose each blog post into 5-7 platform-specific social media posts that drive traffic back to the blog. Each snippet is formatted for the specific platform's best practices, with appropriate hashtags, length, and visual style.

## Platform-by-Platform Playbook

### 1. Instagram — Carousel Posts
**Best for:** Visual destinations, listicles, tips

**Format:** 5-10 slide carousel
```
Slide 1: Title card (blog title as image text)
Slide 2: Key stat or surprising fact
Slide 3-7: One tip/fact per slide (with relevant image)
Slide 8: CTA card ("Link in Bio")
```

**Caption Template:**
```
[Compelling hook — 1-2 lines]

[3-4 bullet points with key takeaways]

[Question to drive engagement — "Which one would you try?"]

✨ Save this for your [destination] trip!

#TravelIndia #IndianTraveler #[Destination]Travel #TravelTips
#BudgetTravel #SoloTravelIndia #BTWVisas

🔗 Full guide in bio!
```

**Optimal posting time:** 6-9 PM IST (weekdays), 10 AM-12 PM IST (weekends)

---

### 2. Pinterest — Vertical Pins
**Best for:** Evergreen travel content, destination guides, itineraries

**Format:** 1000×1500px vertical pins
```
Pin 1: "X Things to Do in [Destination] — Travel Guide for Indians"
Pin 2: "[X]-Day Itinerary: [Destination] — Save This!"
Pin 3: "Cost of [Destination] Trip — Budget Breakdown for Indians"
Pin 4: Infographic pin (budget, checklist, comparison)
Pin 5: Quote pin (pull-out quote from blog)
```

**Pin Description Template:**
```
[Keyword-rich title]

Planning a trip to [Destination]? Here's everything Indian travelers need to know:
- [Bullet 1 — key tip]
- [Bullet 2 — key tip]
- [Bullet 3 — key tip]

Save this pin for later and visit the link for the full guide!

#TravelGuide #IndianTraveler #[Destination]Travel #TravelTipsIndia
#BudgetTravel #SoloTravel #TravelItinerary
```

**Best boards to submit to:** "Travel India", "Indian Travel Bloggers", "Asia Travel", "Budget Travel", "[Destination] Guide"

---

### 3. Twitter/X — Thread
**Best for:** Breaking travel news, tips listicles, personal stories

**Tweet 1 (Hook):**
```
[Surprising fact or hook] 🧵

Example:
"I spent 7 days in Bali on ₹25,000. Here's exactly how I did it 🧵👇"
```

**Tweets 2-7 (Content):**
```
2/ Each item from the listicle or key point from the blog
3/ Keep each tweet self-contained
4/ Use line breaks for readability
5/ Include one emoji per tweet max
...
```

**Final Tweet (CTA):**
```
[Summary tweet]

Full guide with costs and tips 👇
[Link to blog]

Follow @BTWVisas for more travel tips for Indians! ✈️
```

**Best posting time:** 8-10 AM IST (morning scroll), 7-9 PM IST (evening)
**Hashtags:** Use 1-2 max (#TravelTips, #[Destination])

---

### 4. LinkedIn — Professional/Inspirational Post
**Best for:** Travel stories with lessons, cultural insights, work-travel balance

**Format:**
```
[Professional, slightly formal hook]

[2-3 paragraphs — story or insight from the blog]

[1-2 key learnings that apply to professional life]

Question for engagement:
"Have you visited [Destination]? What was your experience?"

[Link to blog in comments or first comment]

#Travel #ProfessionalGrowth #IndianTraveler #WorkLifeBalance
```

**Tone:** More polished, focus on learnings/growth, avoid overly casual language

---

### 5. Facebook — Community-Focused Post
**Best for:** Family travel content, community discussions, longer-form engagement

**Format:**
```
[Relatable hook — Indian family context]

[2-3 paragraphs with personal tone]

Question for the group:
"Would you visit [Destination] with your family? Why or why not?"

[Link to blog]
```

**Groups to cross-post:**
- Indian Travel Groups (request permission first)
- Solo Female Travelers India
- Family Travel India
- Budget Travel India

## Per-Blog Social Snippet Extraction Template

For each blog post, extract:

```

### Snippet 1: Instagram Carousel (8 slides)
[Slide-by-slide content plan]

### Snippet 2: Pinterest (4 pins)
[Pin titles and descriptions]

### Snippet 3: Twitter Thread (5-8 tweets)
[Tweet-by-tweet content]

### Snippet 4: LinkedIn Post (1 post)
[Full post content]

### Snippet 5: Facebook Post (1 post)
[Full post content + group suggestions]
```

## Hashtag Strategy

### Primary Hashtags (Use on every post)
#IndianTraveler #TravelTipsIndia #BTWVisas

### Destination-Specific (Mix 2-3)
#[Destination]/ #[Destination]Travel / #[Destination]Guide

### Content-Specific (Mix 2-3)
#SoloTravel #FamilyTravel #BudgetTravel #LuxuryTravel
#TravelGuide #TravelItinerary #PackingTips #TravelHacks

### Trending (Check before posting)
Check current travel trends on each platform

### Hashtag Count by Platform
| Platform | Recommended Hashtags |
|----------|---------------------|
| Instagram | 10-15 (mix of sizes) |
| Pinterest | 5-10 (keyword-focused) |
| Twitter/X | 1-2 max |
| LinkedIn | 3-5 (professional) |
| Facebook | 2-3 |

## Output
A complete social media repurposing pack for one blog post containing:
- 1 Instagram carousel plan (8 slides with content per slide)
- 4 Pinterest pin ideas (titles + descriptions)
- 1 Twitter thread (5-8 tweets)
- 1 LinkedIn post
- 1 Facebook post
- Platform-specific hashtag sets
- Optimal posting times
