# B06: Budget Breakdown & Cost Analysis

## Objective
Produce transparent, detailed cost analysis blog posts that help Indian travelers budget accurately for their trips. Every figure in INR. Include hidden costs, seasonal variations, and money-saving strategies.

## Budget Blog Types

| Type | Description | Best For |
|------|------------|----------|
| **Destination Cost Guide** | Complete cost of an X-day trip | First-time visitors |
| **Itemized Breakdown** | Deep dive into one category (e.g., "How much does food cost in Bali?") | Budget travelers |
| **Budget Comparison** | X vs Y destination cost comparison | Decision-stage travelers |
| **Seasonal Cost Analysis** | How costs change by season | Flexible travelers |
| **"Hidden Costs" Reveal** | Unexpected expenses Indians don't anticipate | All travelers |
