# Topic Cluster & Pillar Strategy — Visa Content

## Objective
Organize visa content into a structured topic cluster model. This builds topical authority (critical for EEAT), improves internal linking, and signals expertise to both Google and AI Overviews.

## Pillar Page Structure

### What Makes a Good Pillar Page?
- **Broad topic**: Covers an entire visa category (e.g., "Canada Visas for Indians")
- **Comprehensive**: 3000-5000 words overview
- **Hub of links**: Links OUT to all cluster content AND cluster pages link BACK
- **Table of contents**: With jump links to each section
- **Summary sections**: Brief overview of each subtopic (linked to full cluster page)
- **FAQ section**: 10-15 broad questions

### Pillar Page Skeleton
```markdown
# Canada Visa Guide for Indians 2026

## Quick Overview (comparison table of all visa types)
## Canada PR Visa (→ link to cluster page)
## Canada Student Visa (→ link to cluster page)
## Canada Tourist / Visitor Visa (→ link to cluster page)
## Canada Work Visa (→ link to cluster page)
## Canada Business Visa (→ link to cluster page)
## Frequently Asked Questions
## Conclusion + CTA
```

## Cluster Content Structure

### What Makes Good Cluster Content?
- **Specific subtopic**: Focus on ONE visa type or aspect
- **Comprehensive**: 1500-2500 words, fully covering the subtopic
- **Bidirectional linking**: Links TO pillar AND back FROM pillar
- **Independent value**: Can rank on its own for specific queries
- **Schema ready**: FAQ schema, HowTo schema where applicable

### Cluster Page Skeleton
```markdown
# Canada PR Visa for Indians: Complete Guide 2026

## What is Canada PR? (definition → AI Overview target)
## Eligibility Criteria
## Step-by-Step Application Process
## Documents Required (checklist)
## Fees & Processing Time (INR table)
## Success Tips for Indian Applicants
## [Pillar Page: Canada Visa Guide for Indians 2026]
## Frequently Asked Questions
## Conclusion + CTA
```

## Internal Linking Architecture

```
Pillar Page (Topic Authority)
    ↕ (bidirectional links)
Cluster Pages (Subtopic Authority)
    ↕ (bidirectional links)  
Sub-Cluster / Blog Posts (Niche Authority)
```

### Link Density Recommendations
| Page Type | Inbound Links | Outbound Links |
|-----------|--------------|----------------|
| Pillar | From: all cluster pages + homepage | To: all cluster pages |
| Cluster | From: pillar + related clusters | To: pillar + related clusters + sub-clusters |
| Sub-cluster | From: parent cluster + related posts | To: parent cluster + pillar |
