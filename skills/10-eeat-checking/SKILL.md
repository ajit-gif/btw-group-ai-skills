# EEAT SEO Checking — Experience, Expertise, Authority, Trust

## Objective
Audit and enhance visa content to meet Google's EEAT standards. EEAT is a core ranking system — content that lacks EEAT signals will not rank well, especially in AI Overviews where Google prefers authoritative sources.

## The Four EEAT Pillars for Visa Content

| Pillar | Definition | Visa Content Application |
|--------|-----------|-------------------------|
| **Experience** | First-hand or life experience | Show real visa application experience, client stories, case studies |
| **Expertise** | Knowledge and skill in the field | Visa expert authors, verified credentials |
| **Authoritativeness** | Recognized as a go-to source | Backlinks, mentions, industry recognition |
| **Trustworthiness** | Accurate, honest, transparent | Cite official sources, clear disclaimers, honest about success rates |

## EEAT Audit Checklist

### 1. Experience Signals
```markdown
- [ ] Author has applied for / assisted with this visa type
- [ ] Real client testimonials / success stories included
- [ ] Specific, detailed process knowledge (not generic)
- [ ] Personal anecdotes from real visa applications (anonymized)
- [ ] Case studies with actual timelines and outcomes
- [ ] "We have helped X Indian applicants get this visa" (with number)
- [ ] Screenshots or examples of real application forms (redacted)
```

### 2. Expertise Signals
```markdown
- [ ] Author bio with credentials (RCIC, immigration lawyer, MARA agent, etc.)
- [ ] Author byline visible on the page
- [ ] Author LinkedIn / professional profile link
- [ ] Years of experience in visa/immigration field
- [ ] Number of successful applications handled
- [ ] Industry certifications displayed
- [ ] Published in reputable immigration/media publications
- [ ] Speaking engagements, webinars, or workshops conducted
- [ ] Pages include "Reviewed by [Expert Name], [Credentials]"
```


### 3. Authoritativeness Signals
```markdown
- [ ] Site has backlinks from .gov / .edu / reputable domains
- [ ] Cited by other websites as a source (Google Search: site:btwvisas.com)
- [ ] Featured in media / press (Google News mention)
- [ ] Social proof — followers, engagement on visa content
- [ ] Google Knowledge Panel for brand (if applicable)
- [ ] Industry awards or recognition
- [ ] Longevity — how long has the site been operating
- [ ] Guest posts on reputable immigration sites
- [ ] Consistent NAP (Name, Address, Phone) across web
- [ ] Business registered with legitimate authorities
```

### 4. Trustworthiness Signals (Critical for Visa Content)
```markdown
- [ ] ALL claims backed by official sources (.gov, embassy, official immigration sites)
- [ ] Every statistic includes citation with working link
- [ ] Clear disclaimer: "This is informational content, not legal advice"
- [ ] No exaggerated claims ("100% guarantee", "instant approval")
- [ ] Processing times stated with realistic ranges, not guarantees
- [ ] Success rates stated honestly (not "99% success" without proof)
- [ ] Refund/guarantee policy clearly stated (if offering services)
- [ ] Privacy policy and terms of service linked
- [ ] Contact information (email, phone, address) prominently displayed
- [ ] SSL certificate valid (HTTPS)
- [ ] No affiliate links disguised as recommendations
- [ ] User reviews from verified clients (Google Reviews, Trustpilot)
- [ ] Content review date clearly shown
- [ ] Professional indemnity insurance mentioned (if applicable)
```

## Content-Level EEAT Check

### Content Accuracy
```markdown
- [ ] All fees are current (within 3 months)
- [ ] Processing times reflect current reality
- [ ] Policy references match current immigration rules
- [ ] No outdated information (e.g., pre-2026 rules)
- [ ] Forms and links point to correct, current URLs
- [ ] Information is specific — not vague or generic
```

### Content Transparency
```markdown
- [ ] Affiliate relationships disclosed
- [ ] Sponsored content labeled
- [ ] If recommending a product/service, disclose relationship
- [ ] Comparison content is fair and balanced
- [ ] Negative aspects of visas mentioned (not just benefits)
- [ ] Risks and challenges honestly addressed
```

### User Engagement Trust
```markdown
- [ ] Comments enabled with real responses
- [ ] Questions answered promptly (show engagement)
- [ ] "Was this helpful?" feedback widget
- [ ] User-generated success stories
- [ ] Community forum or Q&A section
```

## Step-by-Step EEAT Audit

### 1. Pre-Audit (Page Level)
```markdown
- Read the entire page for accuracy
- Check all external links (are they working? Current?)
- Verify all claims with official sources
- Note any outdated or incorrect information
```

### 2. Score Each Pillar (1-10)
```markdown
| Pillar | Current Score | Target Score | Gap Analysis |
|--------|--------------|--------------|--------------|
| Experience | /10 | 8+ | _________ |
| Expertise | /10 | 8+ | _________ |
| Authority | /10 | 7+ | _________ |
| Trust | /10 | 9+ | _________ |
| **EEAT Total** | **/40** | **32+** | |
```

### 3. Action Items
Based on gaps, create specific action items:
```markdown
Example:
- Gap: No author bio → Action: Add author bio with RCIC credentials
- Gap: No client testimonials → Action: Add 3 verified client reviews
- Gap: Claims without citations → Action: Add official source links to each claim
```

### 4. Implement Fixes
```markdown
- Add EEAT elements (author bio, disclaimers, citations)
- Update outdated information
- Add trust badges / certifications
- Include social proof elements
```

### 5. Re-Audit
After fixes, re-score to ensure targets met.

## Output
An EEAT audit report containing:
- Per-pillar scores (Experience, Expertise, Authority, Trust)
- Specific gaps identified with action items
- Updated EEAT elements added to the page
- Pre- and post-audit comparison
- Re-audit score (target: 32+/40)
