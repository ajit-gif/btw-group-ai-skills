# Grammar Check & Plagiarism Check — Visa Content

## Objective
Eliminate all grammatical errors, spelling mistakes, and stylistic issues. Ensure 100% original content with 0% plagiarism. For visa content, accuracy and professionalism directly impact trust — errors signal unreliability.

## Grammar Check Process

### Step 1: Automated Grammar Check
Use these tools in order:

| Tool | Purpose | When to Use |
|------|---------|-------------|
| **Grammarly** | Full grammar, spelling, tone, clarity | Primary check |
| **Hemingway App** | Readability, passive voice, adverb density | Secondary check |
| **LanguageTool** | Open-source alternative, Indian English support | Tertiary check |
| **Microsoft Word Editor** | Final polish (.docx output) | Final format check |

### Step 2: Manual Review Checklist

**Errors to Watch For in Visa Content:**
```markdown
Common Grammar Issues:
- [ ] Subject-verb agreement ("The requirements is..." → "are")
- [ ] Article usage ("a" vs "an" vs "the" — especially before acronyms)
- [ ] Preposition errors ("apply on" → "apply for", "depend in" → "depend on")
- [ ] Tense consistency (present tense for ongoing facts, past for historical)
- [ ] Comma splices and run-on sentences
- [ ] Apostrophe misuse (it's vs its, applicant's vs applicants')
- [ ] Missing articles before countable nouns
- [ ] "Its" vs "It's" confusion
- [ ] "Your" vs "You're" — common in rushed content
- [ ] Comma in compound sentences before "and"/"but"

Spelling Issues (Indian vs British vs American):
- [ ] Use British/Indian spelling for consistency:
  ✓ "centre" not "center" (unless quoting US-specific content)
  ✓ "programme" not "program" (except computer program)
  ✓ "colour" not "color"
  ✓ "organise" not "organize" (or stay consistent with -ize)
  ✓ "travelling" not "traveling"
  ✓ "licence" (noun) vs "license" (verb)
  ✓ "practice" (noun) vs "practise" (verb)
```

### Step 3: Professional Tone Check
```markdown
Tone Issues Specific to Visa Content:
- [ ] Overly casual ("Hey guys" → "Dear applicants")
- [ ] Overly formal (use "you" not "the applicant")
- [ ] Passive voice > 10% (rewrite in active voice)
- [ ] Jargon without explanation
- [ ] Complex sentences (split into 2-3 shorter ones)
- [ ] Redundant phrases ("advance planning" → "planning")
- [ ] Weak verbs ("make a decision" → "decide")
- [ ] Overused words ("utilize" → "use", "commence" → "start")
```

### Step 4: Consistency Check
```markdown
Consistency Across the Guide:
- [ ] Visa type names capitalized consistently ("Permanent Residence" vs "permanent residence")
- [ ] Acronyms defined on first use (CRS, PNP, ITA, AOR, etc.)
- [ ] Numbers formatted consistently (1,000 or 1000? $160 or 160 USD?)
- [ ] Date format consistent (29 May 2026 or May 29, 2026?)
- [ ] Bullet point punctuation consistent (all end with period or none)
- [ ] Heading capitalization consistent (Title Case or Sentence case)
- [ ] INR formatting consistent (₹12,500 or INR 12,500?)
- [ ] Bold/italic usage consistent
```

### Step 5: Final Human Read
Read the entire guide OUT LOUD. This catches:
```markdown
- Awkward phrasing
- Missing words
- Run-on sentences
- Unnatural keyword placement
- Tone inconsistencies
- Flow issues between sections
```

## Plagiarism Check Process

### Acceptable Plagiarism Scores
| Score | Verdict | Action |
|-------|---------|--------|
| 0-2% | ✅ Clean | No action needed |
| 2-5% | ✅ Acceptable | Review flagged sources, ensure proper citation |
| 5-10% | ⚠️ Warning | Rewrite flagged sections |
| 10%+ | ❌ Fail | Major rewrite required |

### Tools to Use
```markdown
Free:
- Duplichecker.com (1000 words/day free)
- SmallSEOTools Plagiarism Checker
- Quetext (free tier)
- Prepostseo

Paid (Recommended):
- Copyscape (industry standard for web content)
- Grammarly Premium (includes plagiarism check)
- Turnitin (for higher education contexts)
```

### Common Plagiarism Risks in Visa Content
```markdown
⚠️ HIGH RISK: Copying official requirement text from .gov websites
  → Solution: Paraphrase thoroughly, cite the source

⚠️ HIGH RISK: Using competitor's content structure and phrasing
  → Solution: Different structure, unique examples, BTWVisas voice

⚠️ MEDIUM RISK: Boilerplate policy descriptions
  → Solution: Add commentary, Indian context, examples

⚠️ MEDIUM RISK: Fee tables copied from official sources
  → Solution: Present in unique format, add INR conversions, add notes

⚠️ LOW RISK: Common visa terminology (cannot plagiarize generic terms)
  → No action needed
```

### Paraphrasing Guidelines
```markdown
DO:
✓ Change sentence structure (active ↔ passive, order of information)
✓ Use synonyms (but keep terminology accurate for visas)
✓ Change examples to India-specific ones
✓ Add your own analysis/commentary
✓ Combine multiple sources into unique synthesis
✓ Cite the official source as reference

DON'T:
✗ Just swap words with a thesaurus (especially for legal/visa terms)
✗ Keep the same sentence structure with different words
✗ Remove necessary technical accuracy for the sake of "uniqueness"
✗ Fail to cite direct quotes from official sources
```

## Final Quality Gates

### Before Output
```markdown
Grammar Gate:
- [ ] Automated grammar check passed (Grammarly score: 90+)
- [ ] Readability: Grade 8-10 (Hemingway)
- [ ] 0 spelling errors
- [ ] Active voice used in 90%+ of sentences
- [ ] Adverb count < 5% of words
- [ ] Consistency check passed (see checklist above)

Plagiarism Gate:
- [ ] Copyscape score < 5%
- [ ] All flagged matches reviewed and either rewritten or properly cited
- [ ] No direct copying from official sources without citation
- [ ] All statistics/facts attributed to sources
- [ ] Unique structure (not mirroring competitor pages)
```

## Output
A quality report with:
- Grammar scores (Grammarly, Hemingway)
- All corrections applied (before/after for key errors)
- Plagiarism check score and flagged URLs
- Rewritten sections (if plagiarism found)
- Consistency checklist completion
- Final human-read sign-off
