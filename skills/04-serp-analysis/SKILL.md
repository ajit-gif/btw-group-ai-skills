# SERP Analysis — Visa Keywords

## Objective
Analyze the Google Search Engine Results Page (SERP) for your target visa keyword to understand:
- Which SERP features are present
- What content formats Google rewards
- Featured snippet / AI Overview opportunities
- "People Also Ask" questions to answer
- Ranking factors for top positions

## SERP Features to Track for Visa Content

```
Standard Results ─── Blue links with organic rankings
Featured Snippet ─── Position 0 (definition, table, list, step) ← HIGH PRIORITY
AI Overview ──────── Google AI-generated summary ─── HIGHEST PRIORITY
People Also Ask ─── Expandable Q&A box ─── MUST TARGET
Knowledge Panel ─── Right-side info card (for countries/visa types)
Image Pack ───────── Image results (visa documents, forms)
Video Carousel ───── YouTube videos (visa guides, interviews)
Top Stories ──────── News articles (visa policy changes)
Local Pack ───────── Visa consultants near me (local SEO)
Related Searches ─── Bottom of SERP keywords
```

## Step-by-Step SERP Analysis

### 1. Search in Incognito + Indian Location
```powershell
# Search with location set to India
Google search → Settings → Location → India
# OR use tools like:
- Wincher.com (SERP checker)
- Ahrefs SERP Checker
- Manual incognito with VPN set to India
```

### 2. Document SERP Features
Create a SERP features checklist:

| Feature | Present? | Content Opportunity |
|---------|----------|-------------------|
| AI Overview | Yes/No | Optimize definition + steps |
| Featured Snippet | Yes/No | Target position 0 format |
| People Also Ask (count) | ___ questions | Answer all in FAQ |
| Image Pack | Yes/No | Create infographics |
| Video Results | Yes/No | Create video guide |
| News Box | Yes/No | Monitor policy changes |
| Related Searches | ___ keywords | Topic expansion ideas |

### 3. Analyze Top 10 Ranking Pages
For each of the top 10 results, document:

| Metric | Page 1 | Page 2 | ... | Pattern |
|--------|--------|--------|-----|---------|
| Domain Authority | | | | Higher DA = easier to rank? |
| Page word count | | | | Longer content better? |
| Content format | | | | Guide vs list vs step |
| Heading structure | | | | H1, H2, H3 pattern |
| Number of images | | | | Visual content correlation |
| Internal links | | | | Linking strategy |
| External links | | | | .gov citations |
| Social signals | | | | Shares, comments |
| Page speed (mobile) | | | | Core Web Vitals |
| Last updated date | | | | Freshness signal |
| Schema markup | | | | FAQ, HowTo, Article |

### 4. Featured Snippet Analysis (for AI Overview Optimization)
```
Current featured snippet content:
┌─────────────────────────────────┐
│ [Extract the exact text]         │
│ Format: Paragraph / List / Table │
│ Source domain: __________        │
│ Word count: ______               │
└─────────────────────────────────┘

→ Can we create BETTER content for this snippet?
→ Can we add a definition that answers "What is X?" concisely?
```

### 5. People Also Ask (PAA) Extraction
Extract ALL PAA questions from the SERP:

```
1. What is the minimum bank balance for [visa type]?
2. How long does [visa] processing take?
3. Can I work on a [visa type] in [country]?
4. What documents are required for [visa]?
5. Is [visa] difficult to get for Indians?
...
```

→ **Every PAA question must be answered in the content** (use FAQ section + in-text answers)

### 6. AI Overview Specific Analysis
If AI Overview is present for the query:

```
AI Overview Content:
- Source URLs cited: _______
- Format (paragraph/list/table): _______
- Key claims made: _______
- Entities mentioned: _______
- Tone (neutral/official/helpful): _______

Strategy:
→ Can we match or exceed the cited sources in EEAT?
→ Can we provide MORE current information?
→ Is there a gap in the AI Overview we can fill?
```

## Output
A SERP analysis document containing:
- Full SERP features inventory
- Top 10 competitor page analysis table
- Featured snippet content extraction
- All PAA questions (categorized by topic)
- AI Overview analysis (if present)
- Related searches list
- Content opportunity gaps identified
- Recommended content format for the query
