# Keyword Research — Visa Content

## Objective
Identify high-value, searchable keywords that Indian users use when researching visa applications. Target keywords optimized for both traditional search and AI Overview/LLM extraction.

## Tools to Use
- Google Keyword Planner
- Ahrefs / SEMrush (if available)
- Google Autocomplete / "People Also Ask"
- AnswerThePublic (for question-based keywords)
- ChatGPT / Gemini (for keyword clustering)
- Google Trends (seasonality check for visa types)

## Step-by-Step Process

### 1. Seed Keywords
Start with these seed keyword categories for visa content:

| Category | Examples |
|----------|----------|
| **[Country] visa for Indians** | "Canada visa for Indians", "UK tourist visa from India" |
| **[Visa type] requirements** | "Student visa requirements for Canada", "Work visa eligibility" |
| **Application process** | "How to apply for US visa from India", "Step by step visa application" |
| **Fees & processing** | "Australia visa fee for Indians", "Schengen visa processing time" |
| **Eligibility** | "Who can apply for Japan visa", "Income requirement for UK visa" |
| **Success stories** | "India to Canada visa success stories", "Visa approval tips" |
| **Document checklist** | "Documents required for US visa", "UK visa document checklist" |
| **Interview preparation** | "Visa interview questions and answers", "US visa interview tips" |
| **Track application** | "How to check visa application status", "Visa tracking India" |
| **Rejection reasons** | "Visa rejection reasons for Indians", "How to avoid visa rejection" |

### 2. Keyword Expansion
For each seed keyword, expand using:
- **Google Autocomplete**: Type seed in incognito → capture suggestions
- **People Also Ask**: Scrape PAA boxes from SERP
- **Related Searches**: "Searches related to [keyword]" at SERP bottom
- **LSI Keywords**: Latent Semantic Indexing keywords via tools

### 3. Keyword Filtering Criteria (Indian Audience)
| Criterion | Threshold |
|-----------|-----------|
| Search Volume (monthly) | ≥ 100/month for niche, ≥ 1000/month for head terms |
| Keyword Difficulty (KD) | ≤ 40 for new content, ≤ 60 if site has authority |
| Intent Match | Informational (60%) + Commercial (40%) |
| CPL (Cost Per Lead) | Not applicable for organic |
| Indian Search Intent | Contains "for Indians", "from India", "in rupees", etc. |
| AI Overview Potential | High if keyword triggers PAA, featured snippets, definition queries |

### 4. Keyword Clustering
Group keywords into topic clusters:

```
Pillar Keyword: "Canada PR visa for Indians"
└── Cluster:
    ├── "Canada Express Entry requirements for Indians"
    ├── "CRS score calculator for India"
    ├── "Canada PR processing time from India"
    ├── "Canada PR fee in rupees"
    └── "Canada PR vs work visa which is better"
```

### 5. Question-Based Keywords (Crucial for AI Overviews)
AI Overviews heavily favor question-answer formats. Prioritize:

| Question Type | Examples |
|---------------|----------|
| **What** | "What is the minimum bank balance for Canada student visa?" |
| **How** | "How to apply for US B1/B2 visa from India?" |
| **Which** | "Which visa type is best for Indian students?" |
| **When** | "When is the best time to apply for Schengen visa?" |
| **Can** | "Can I work in UK on a tourist visa?" |
| **Is** | "Is IELTS required for Canada PR from India?" |

### 6. Keyword Prioritization Matrix
```
High Volume + Low Difficulty = 🟢 TOP PRIORITY (Create now)
High Volume + High Difficulty = 🟡 STRATEGIC (Build authority first)
Low Volume + Low Difficulty = 🔵 QUICK WINS (Use as cluster content)
Low Volume + High Difficulty = ⚪ AVOID (Not worth effort)
```

## Output
A keyword research document with:
- Primary keyword (1-2)
- Secondary keywords (5-8)
- Long-tail / question keywords (10-15)
- LSI keywords (5-10)
- Keyword cluster map
- Search intent classification per keyword

## Quality Check
- [ ] All keywords verified with actual search volume
- [ ] Keywords include "for Indians" / "from India" variants
- [ ] Question-based keywords identified for AI Overview targeting
- [ ] Intent classification completed (informational vs commercial)
- [ ] Competitor gap keywords identified
