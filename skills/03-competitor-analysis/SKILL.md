# Competitor Content Analysis — Visa Guides

## Objective
Analyze top 5-10 competitor pages ranking for your target visa keyword. Identify content gaps, structural patterns, and differentiation opportunities to outrank them in both traditional search and AI Overviews.

## Competitors to Analyze

### Direct Competitors (Visa Service Websites)
- btwvisas.com (your own site — benchmark)
- visaguru.com
- visafirst.com
- y-axis.com
- immigration.ca / canadavisa.com
- idp.com (for student visas)

### Indirect Competitors (Content/Media)
- travel.gc.ca (official Canada)
- uk.gov (official UK)
- uscis.gov (official US)
- timesofindia.indiatimes.com (Indian news)
- reddit.com/r/visas (community)
- quora.com (Q&A platform — important for AI Overviews)

### AI Competitors
- ChatGPT / Gemini outputs for the same query
- Google AI Overview snippets already ranking

## Analysis Framework

### 1. Content Structure Analysis
| Element | Your Page | Competitor 1 | Competitor 2 | Best Practice |
|---------|-----------|--------------|--------------|---------------|
| H1 tag | | | | Include primary keyword |
| Word count | | | | Target 2000-3000 words |
| Table of Contents | | | | Yes (with jump links) |
| FAQ section | | | | 10-15 questions minimum |
| Step-by-step guide | | | | Numbered steps |
| Document checklist | | | | Printable format |
| Fee table (INR) | | | | Convert to INR |
| Processing time | | | | Current dates |
| Success stories | | | | Real testimonials |
| Expert quote/signoff | | | | For EEAT |

### 2. Keyword Gap Analysis
```
Competitor Keywords (they rank for) ∩ Your Keywords (you rank for) = Overlap
Competitor Keywords - Your Keywords = GAP → Create content here
Your Keywords - Competitor Keywords = ADVANTAGE → Defend/enhance
```

### 3. AI Overview Gap Analysis
| Check | Questions |
|-------|-----------|
| Does this query trigger AI Overview? | `site:google.com "keyword"` |
| What format does AI Overview use? | Definition? Steps? Table? List? |
| Which source does AI Overview cite? | Official (.gov) or media? |
| Can we be the cited source? | Need FAQ schema, short definitions, steps |
| Is there a "People also ask" section? | Extract all PAA questions → answer in content |

### 4. EEAT Comparison
| EEAT Factor | Competitor Score (1-10) | Your Target |
|-------------|------------------------|-------------|
| Author expertise | Medical/legal disclaimer? Visa expert bio? | Author bio with credentials |
| Site authority | Domain rating, backlinks | Improve with .gov backlinks |
| Content accuracy | Outdated vs updated | Regular update cycle (quarterly) |
| User trust | Reviews, testimonials | Indian client testimonials |
| Official citations | Links to .gov sources | Cite immigration websites |
| Content freshness | Last updated date | Show "Last updated: May 2026" |

### 5. Content Quality Scoring Matrix
| Criteria | Weight | Competitor | Your Target |
|----------|--------|------------|-------------|
| Readability (Flesch score) | 10% | | Grade 8-10 |
| Mobile optimization | 15% | | Pass Lighthouse mobile |
| Internal linking | 10% | | 3-5 relevant links |
| External authority links | 15% | | 5+ .gov links |
| Multimedia (videos, images) | 10% | | 2+ visuals per section |
| Freshness (last updated) | 10% | | < 3 months old |
| Schema markup | 10% | | FAQ + HowTo + Article |
| User engagement signals | 10% | | Comments, shares |
| Indian localization | 10% | | INR, local terms, cultural context |

## Step-by-Step Execution

1. **Identify top 10 ranking URLs** for target keyword (use incognito Google)
2. **Extract each page's structure** — H1, H2s, H3s, word count, media
3. **Note content formats** — lists, tables, videos, FAQs, calculators
4. **Identify featured snippets** — does competitor have position 0?
5. **Check AI Overview triggers** — does this query show AI Overview?
6. **Analyze backlinks** — what links do competitors have that you don't?
7. **Read user comments** — what questions are users asking? (gap opportunity)
8. **Score each competitor** on the EEAT comparison matrix
9. **Document your differentiation strategy**

## Output
A competitor analysis document containing:
- Top 5 competitor URLs with their content stats
- Keyword gap analysis (what they have that you don't)
- Content structure comparison table
- AI Overview / featured snippet opportunities
- EEAT gap analysis
- Differentiation strategy (3-5 unique angles)
- Content improvement checklist
