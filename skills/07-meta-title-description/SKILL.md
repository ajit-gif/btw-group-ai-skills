# Meta Title & Meta Description — Visa SEO

## Objective
Craft compelling, SEO-optimized meta titles and descriptions that drive clicks from Indian users and satisfy Google's requirements for featured snippets and AI Overview citation.

## Meta Title Structure

### Formula
```
Primary Keyword | Secondary Keyword | [Country/Year] — Brand
```

### Pattern for Visa Content
```markdown
[Visa Type] Visa for Indians [Year]: [Benefit/Key Info] — [Brand]

Examples:
✓ Canada PR Visa for Indians 2026: Eligibility, Fees & Process — BTWVisas
✓ US Student Visa (F1) Guide 2026: Requirements for Indian Students — BTWVisas
✓ UK Tourist Visa from India: Fees, Documents & Processing Time 2026 — BTWVisas
```

### Rules
| Rule | Description |
|------|-------------|
| Length | 50-60 characters (Google displays ~60 chars) |
| Primary keyword | Include at the BEGINNING |
| Year | Add "2026" for freshness signal |
| Location | Include "for Indians" or "from India" |
| Brand | End with "— BTWVisas" or "| BTWVisas" |
| Unique | No duplicate titles across the site |
| Compelling | Include benefit or key info (eligibility, fees, process) |
| Match content | Title must reflect actual page content |
| Avoid clickbait | Be accurate — misleading titles hurt EEAT |

### Title Patterns by Intent
| Intent | Pattern | Example |
|--------|---------|---------|
| Informational | [Topic] for Indians [Year]: Complete Guide | "Canada PR for Indians 2026: Complete Guide" |
| Commercial | [Topic] for Indians [Year]: Fees, Eligibility & Process | "Australia Student Visa 2026: Fees & Eligibility" |
| Transactional | How to Apply for [Topic] from India [Year]: Step-by-Step | "How to Apply for US Visa from India 2026" |
| Comparison | [Topic A] vs [Topic B] for Indians [Year]: Which is Better? | "Canada PR vs Work Visa 2026: Which is Better?" |
| Urgent | [Topic] 2026: Latest Updates & Changes for Indians | "UK Visa Rules 2026: Latest Updates for Indians" |

## Meta Description Structure

### Formula
```
[Action verb] + [keyword] + [key details/benefits] + [what user will learn] + [CTA/trust signal]
```

### Rules
| Rule | Description |
|------|-------------|
| Length | 150-160 characters |
| Primary keyword | Include naturally (once, preferably early) |
| Secondary keyword | Include if possible |
| Value proposition | What will the reader learn/get? |
| CTA | Subtle call-to-action ("Learn how", "Discover", "Check eligibility") |
| Trust signal | "Updated 2026", "Official sources", "Expert guide", "1000+ success stories" |
| No quotes | Don't use "quotes" — Google sometimes truncates at quote marks |
| Unique | Every page needs a unique description |
| Match search intent | If query is "how to apply", description should mention step-by-step |

### Description Patterns by Intent
```markdown
Informational:
"Planning to apply for [Country] [Visa Type]? Our 2026 guide covers [key topics]. Get expert tips, document checklists & success stories for Indian applicants."

Commercial:
"Compare [Visa A] vs [Visa B] for Indians. We break down fees [INR], processing time, eligibility & success rates. Find the best option for you."

Transactional:
"Step-by-step guide to applying for [Visa Type] from India. Complete document checklist, fee breakdown & processing timeline. Updated May 2026."

Urgent/News:
"Latest [Country] visa policy changes for Indian applicants in 2026. New fees, rule changes & processing updates — everything you need to know."
```

### Description Examples
```markdown
✓ "Planning to apply for Canada PR from India? Our 2026 guide covers Express Entry, CRS score, fees in INR & step-by-step process. Expert tips for Indian applicants. Updated May 2026." [158 chars — ✓]

```

## AI Overview Optimization for Meta

Since AI Overviews often display meta descriptions as source citations, optimize for this:

```markdown
**AI Overview Friendly Meta Description Structure:**
1. Start with a clear definition or value statement
2. Include the primary question the page answers
3. Use natural language (not keyword-stuffed)
4. Include a statistic or benefit

Example:
"Canada PR visa for Indians: Learn eligibility, CRS score requirements, fees in INR, and step-by-step application process. Complete 2026 guide with expert tips." [character count: 147]
```

## Step-by-Step Creation Process

1. **Research the SERP** — Check current meta titles of top 10 ranking pages
2. **Identify title pattern** — What format do top results use?
3. **Draft 3 title options** — A/B/C with different hooks
4. **Draft 3 description options** — Different angles
5. **Check character counts** — Title ≤60, Description ≤160
6. **Verify keyword inclusion** — Primary keyword in both title and description
7. **Check for uniqueness** — Search site:"btwvisas.com" to avoid duplicates
8. **Get AI Overview check** — Would Google show this as a source?

## Output
For each visa guide:
- 3 meta title options (with character counts)
- 3 meta description options (with character counts)
- Recommended primary + secondary selection
- Current top 10 competitor title/description analysis
