# Content Generation — Visa Guide Writing

## Objective
Generate comprehensive, well-structured visa guide content that:
- Targets Indian audiences (language, context, concerns)
- Balances informational + commercial intent
- Is optimized for AI Overview extraction
- Passes EEAT scrutiny
- Delivers genuine value (not fluff)

## Brand Voice & Tone

### BTWVisas Brand Voice
| Attribute | Description |
|-----------|-------------|
| Tone | Helpful, authoritative, warm but professional |
| Language | Simple English (Grade 8-10 level) |
| Approach | "Guide" not "Sales pitch" — educate first, CTA second |
| Cultural | India-first perspective, understand local concerns |
| Trust | Cite official sources, show expertise |
| Format | Scannable, structured, mobile-optimized |

### DO and DON'T
```
DO:
✓ Use "you" and "your" — direct address
✓ Start with empathy — "We know visa applications can be stressful"
✓ Explain WHY behind each requirement
✓ Use Indian currency (INR) alongside foreign currency
✓ Mention Indian embassy/consulate locations
✓ Include India-specific timelines
✓ Reference popular Indian banks/UPI for fee payments
✓ Use examples Indian readers can relate to

DON'T:
✗ Use jargon without explanation
✗ Assume prior visa knowledge
✗ Write generic content that could be for any country
✗ Use complex sentences (keep Flesch Grade 8-10)
✗ Be overly promotional — educate first
✗ Ignore the emotional aspect of visa applications
```

## Content Generation Rules

### Rule 1: Definition-First Approach (AI Overview Priority)
```markdown
Every major section must start with a clear definition:

✓ "Canada PR (Permanent Residence) is a status that allows you to live, work,
   and study anywhere in Canada. For Indians, it's one of the most popular
   immigration pathways..."

✓ "A UK Student Visa (Tier 4) allows Indian students to study at approved
   UK educational institutions. As of 2026, the visa permits..."
```

### Rule 2: Answer Questions Immediately
```markdown
When a section heading is a question, answer directly in the first sentence:

H2: "What is the minimum bank balance for Canada Student Visa?"
→ "The minimum bank balance for a Canada Student Visa is CAD $20,635
  (approximately ₹12.5 lakhs) as of 2026, plus first year tuition fees."
```

### Rule 3: Use Indian Context Throughout
```markdown
✓ "Apply through VFS Global centers in Delhi, Mumbai, Bengaluru, Chennai..."
✓ "Fees can be paid via credit card, debit card, or UPI..."
✓ "Document notarization must be done at a registered Indian notary..."
✓ "For Indian degrees, you may need an Educational Credential Assessment (ECA)..."
```

### Rule 4: Include Social Proof
```markdown
✓ "BTWVisas has helped over 5,000 Indian professionals get their Canada PR"
✓ "95% of our clients rate their visa experience as 'Excellent'"
✓ "Featured in [Publication] for visa expertise"
```

### Rule 5: Write for Skimming (Mobile-First)
```markdown
- Short paragraphs (2-4 sentences max)
- Bullet points for lists
- Bold key terms
- Numbered steps for processes
- Subheadings every 150-200 words
- Tables for comparison data
- Highlight boxes for important notes
```

## AI Overview Extraction Block (Visible Blockquote)
Place this at the TOP of the page (immediately after H1). Use visible blockquote format, NOT HTML comments.

```markdown
> **What is [Visa Type]?** [2-3 sentence definition]
> 
> **Key 2026 Updates:** [3-4 bullet-style policy changes]
> 
> **Quick Facts:** [Key stats — processing, fee, requirements, centers]
```
